SASS Elements

A set of useful SASS mixins based on lesselements.com.
See on Github.

How to use:
  • Download the "elements.scss".
  • Add to any folder in your project
  • Import it at the top of your stylesheet:
    @import "elements.scss";
Box shadow
@include drop-shadow();
Box shadow with custom properties
@include drop-shadow(4px, 4px, 12px, 0.5);
Box gradient
@include gradient();
Box gradient with custom properties
@include gradient(#444, #222, #000);
